# Khanbam - Reactjs Project

## This project is a simple implementation with ReacJs, using :

- ReactJs (of curse)
- Fetch Api 
- Styled Componnnets
- CSS3
- HTML
- ES6 (of curse)

## The basic concept of this project is:

 - Simulate an Kamban flow.
 - Using drag and drop.
 - And Style similar to Pipefy.
